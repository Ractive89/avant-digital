## Abstract

*托马斯 - 米歇尔* 继罗蒂的语言学转向后，提出图像转向。在 20 世纪维特根斯坦语言哲学中的图像恐惧以及文化领域对图像的下意识抵制中把握到图像转向的症候，并在潘诺夫斯基的基础上重构图像学。米歇尔在图像理论中有后语言学、后符号学的新发现，他阐释了元图像，认为图像如语言一样可以自我呈现，并主张回到图像与语言本身重新思考图像与语言的关系，以此瓦解语言对图像的控制。

## Notes

- 1992 年 *托马斯 - 米歇尔* 在『艺术论坛』首次提出 **图像转向** (pictorial turn)

### 图像学

#### 潘诺夫斯基的图像学

- *潘诺夫斯基*的图像学奠定了图像理论学科化的基础，首次架构图像科学。在解释 *文艺复兴时期* 视觉艺术的主题时，逐步发展的跨学科式解释图像的规范或方法论
	- 图像并非模仿之物，而是特定时期历史文化的象征形式。
	- *潘诺夫斯基*认为图像主题及其包含的历史文化意义的三个层次解读
		- **前图像志**分析，依据实际经验、风格史的知识解释作品描绘的题材
		- **图像志**分析，依据原典知识解释作品表现了什么约定俗成的故事或寓言等
		- **图像学**分析，依据对人类心灵基本倾向的把握，结合文化史知识，解释作品的文化象征意义

#### 米歇尔的图像学

- *米歇尔* 主张要在 *潘诺夫斯基* 的基础上复兴图像学，发展潘氏在艺术史领域中开拓的跨学科研究视野。
	- 把艺术的历史与图像史做了区分，推崇视野更为广阔的图像史。
	- 米歇尔与潘氏的问题视域完全不同
		- 米歇尔: 图像 “涉及言语图像、语言图像、画面图像，而且牵涉到镜像、映像、影像、自然界的整个视觉现象，以及记忆、幻象和认知行为本身这种在别处产生图像的中间领域。认知图像导致想象、幻象、并导致记忆等。”
		- “符号学、修辞艺术、诗学、美学、人类学、精神分析学、伦理学、意识形态批评和艺术史都交织在图像理论、文本、观者、文学描写和叙事等理论中”

- *米歇尔* 的图像学**突破**
	- 突破模仿论、反映论视域下的图像观念 ( 图像视为世界的摹本和意向性 )
	- 突破语言学视域下的图像观念 ( 图像是语言符号的一种 )
- *米歇尔* 的图像研究**回到图像**自身，==对图像的后语言学、后符号学的再发现==
	- 没有采用语言学模式分析图像 ( 遮蔽图像表征的特殊性 ) 
	- 没有继承潘诺夫斯基的图像学 ( 文化研究 ) 
	- 没有延续沃尔夫林的形式主义研究 ( 形式风格研究 ) 

- 图像如语言一样可以自我呈现，它们共同参与对世界的表征

### 图像转向

- 人文科学开始重视图像表征 ( 非语言表征 )，图像成为人文科学的中心话题。
	- *米尔佐夫*: 创造了后现代性的，正是文化的视觉危机，而不是其文本性 ... 视觉对于纯粹从语言学方面界定文化的任何企图都是一种破坏和挑战。[^1]
- 在文化公共领域中，新的图像生产技术制造了**图像景观**，并引发了人类对图像的恐惧与抵制。
	- 人类的图像史中一直潜藏着**惧怕**图像的观念，人们在制造图像的同时，也在制造恐惧。
	- *托马斯 - 米歇尔*: 一方面，视频、赛博技术、电子复制时代以不可预期的能力发展了视觉模拟和幻觉的种种新形式。 另一方面，形象恐惧、担心形象的力量最终可能会摧毁它们的创造者和操纵者的焦虑，与形象制造本身一样历史悠久。

- 德国的 *戈特弗里德 - 伯姆* 也探讨人文科学出现的图像转向 (ikonische Wende/ iconic turn)。

#### Image 形象

- 形象在视觉艺术中都扮演着角色，形象作为图像所表现的内容的名称或它全部形式的整体，也可指语言形象，一种隐喻，一种图形，一种命名的事物或本质，甚至一个文本总体化的常规语言图示。[^2]
- 形象是意识的产物，同一个形象可由不同媒介呈现。

#### Picture 图像

- 图像就是形象加上它的支撑物。[^2] ( 如画、照片、雕塑等 ) 
- 图像的本源性是形象，研究图像的本体性要深入到形象中
- 研究形象在文化实践中的表现形式，要深入到可感的图像世界。

#### Icon 圣像/图像

- 图像解释为符号学家皮尔斯的三种符号类型 Icon、Index、Symbol
- 使用 Icon 为了将理论置于传统图像观念研究中，为重构图像学寻求合法性。

### 图像转向的途径

#### 语言哲学

- 图像转向是**语言学转向**直接发展的结果。 
	- *理查德 - 罗蒂* 把哲学史描述成一系列转向
		- 古代和中世纪的哲学图景是关注事物 ( 本体论 ) 
		- 17 - 19 世纪的哲学图景是关注观念 ( 认识论 ) 
		- 当代的哲学图景是关注词语 ( 语言论 ) 

- *维特根斯坦*、*罗蒂* 等人出于保护语言免受形象的他性干扰而抵制视觉，企图把图像从语言中完全划出，或认定为语言事实，这遮蔽对图像生成意义的关注。
	- *维特根斯坦*的语言哲学的图像恐惧: 图像俘虏了我们，而我们无法逃离它，因为它置于我们的语言之中，而且语言似乎不可阻挡地向我们重复它。
	- *罗蒂*: 把视觉的、尤其是镜像、隐喻一起排除在我们的言辞之外。

- *米歇尔*认为==图像转向随着图像恐惧一同出现==
	- *米歇尔*: 图像转向常常率先被捣毁图像者注意到，他们会表达对一种具有威胁性的‘世界图像’的恐惧和警觉，正如海德格尔，他把‘作为图像的世界’的发端与现代科技理性的统治等量齐观。

- 米歇尔在维特根斯坦的哲学思想体系中获取灵感
	- **逻辑图像论**
		- 语言表述中的命题看作事实的图像，语言的命题具有图像的性质。图像处于连接语言和世界之间的那种内在的图示关系之中。
		- 语言与世界、图像与事实具有同构关系。
	- **语言游戏说**
		- 否认逻辑图像论，主张从灵活多变的日常语言实践出发，在语言使用游戏中体会事实的意义。
			- 概念通过各种语言规范产生意义， 事实的意义取决于出现的背景、语言系统

#### 图像学

- 图像转向复兴了图像学
	- 二战后，潘诺夫斯基的图像学由于抽象艺术兴起导致衰弱。
	- 20 世纪后期，视觉文化的兴起，人文社科发生图像转向。

- 潘诺夫斯基的图像学试图通过逻各斯把握图像，最终图像学发展成了总体化的**抽象视觉史**。
	- 生成于历史文化语境中的图像包含着特定的主题或思想。这种主题和思想是能用言辞表达并获得广泛认同的抽象时代精神。==一个时代的意识形态观念会赋予一个时代的图像一种同质的文化意义==，这种文化意义从一个时代的观念史中抽象出来，又印证到那个时代的艺术作品中。它们最终会在图像的构图、主题、材料、形象中呈现出来。

 - 米歇尔复兴图像学的抵制与超越
	- 潘氏过分概括化的**宏大叙事**
	- 传统图像学的图像解释把他者**同质化**为统一的话语之中

- 复兴图像学的基本主张
	- ==关注由语言和图像共同建构的存在者==
		- 理论研究不能停留在单纯的视觉经验领域或文本分析的模式，而应把研究从已有的认识论的认知背景转换成道德的、政治的和诠释的背景。
	- ==对图像意识形态性的批判认识==
		- 把图像当作视觉性、机器、体制、话语、身体和喻形之间的一种复杂的相互作用。要认识到，观看行为 ( 观看、凝视、浏览以及观察实践、监视与视觉快感 ) 可能与阅读的诸种形式 ( 解密、解码、解释等 ) 是同等深奥的问题，而基于文本性的模式可能难以充分阐释视觉经验或视觉识读。[^3]

### 元图像

- 元图像即关于图像的图像，是一种能**自我指涉**的图像。

- 米歇尔的元图像是==把图像还原为一种图像事实==。
	- **图像不仅仅是可见世界的视觉表征，还是图像自身的表征**
	- “... 不是从艺术或语言派生一个图像自我指涉的模式，而是看图像是否提供了自己的元语言。我想要验证的观念是: 图像也许能够反映自身，能够提供**二级话语**， 告诉我们——至少向我们展示——有关图像的东西。”
	- “我不认为图像具有重要的艺术意义或深刻的哲学意义，它们仅仅呈现了图像反映自身的解释的方式”

- 元图像在不同学科话语的阐述中，成为一种超图像，即架构理论意义的图像。
	- “元图像是一架能够移动的文化机器, 它既可以发挥边缘作用，做插图; 也可以发挥核心作用, 做概括的形象, 我称它囊括了整体知识的超图像，一种知识理论。”

#### 螺旋

ST 1964 New World New Yorker

- 米歇尔的解读
	- 『螺旋』是现代绘画史的一个寓言，即绘画史开始于再现外部世界，终结于表征内部抽象世界。
	- 『螺旋』是现代艺术家的写照，现代艺术家的生命就如同封闭的螺旋一样，越来越窄，排斥了螺旋外的广袤世界，只能靠自己的本质过活。
	- 『螺旋』是观者 “图像转向” 后的世界写照，通过图像生产而建构生成的世界。我们生活在形象的世界里，这个世界除图像之外一无所有。

- 螺旋形状的封闭结构消解了再现论中内部与外部的二元对立, 在这里作品不再是模仿之物或外部世界的影子，而是呈现了绘画本身。
